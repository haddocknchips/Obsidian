# Obsidian Knowledge Vault

Purpose:
Synthesis and long-lived thinking across projects.
  
Not used for:
- Daily notes
- Meetings
- Tasks

Structure:
- Projects: active outcomes
- Areas: ongoing domains
- Resources: reusable concepts and references
- Archive: inactive material

Relationship with vimwiki:
- Vimwiki:
	- Chronological
	- Messy is okay
	- Project-driven
	- Fast
- Obsidian:
	- Curated
	- Sparse
	- Reusable
	- Intentional
Only promote ideas ***worth keeping***

Sync: GitHub only
